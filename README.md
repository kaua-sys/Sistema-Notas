# Sistema de Notas

Projeto desenvolvido em JavaScript para praticar lógica de programação, arrays, estruturas de repetição e estruturas de decisão.

## Funcionalidades

* Adicionar notas
* Calcular média das notas
* Exibir a maior nota
* Exibir a menor nota
* Validar quando não houver notas cadastradas

## Tecnologias Utilizadas

* JavaScript

## Conceitos Praticados

* Variáveis
* Arrays
* Loops (`for`)
* Condicionais (`if`)
* Estrutura de seleção (`switch`)
* Entrada de dados com `prompt`
* Saída de dados com `console.log`

## Como Executar

1. Abra o arquivo JavaScript em um ambiente compatível.
2. Execute o programa.
3. Escolha uma opção no menu.
4. Adicione notas e consulte as informações desejadas.
